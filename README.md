# todo-project

## to use project 
- install node
- install xampp
- start Apache & start MYSQL from XAMPP
- open project folder in terminal then write these commands
```
cd backend
```
```
npm install
```
```
npm run migrate
```
```
npm start
```

- open index.html in browser

## any time after this first one 
```
cd backend
```
```
npm start
```

- open index.html in browser
